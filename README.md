# Aplikasi Database Restoran (CRUD)

Aplikasi ini merupakan aplikasi database restoran berbasis **PHP dan MySQL** yang menerapkan operasi **CRUD (Create, Read, Update, Delete)**. Aplikasi ini digunakan untuk mengelola data menu dan pesanan restoran serta dilengkapi dengan fitur login admin.

## Teknologi yang Digunakan
- PHP
- MySQL
- Apache (XAMPP)
- HTML & CSS

## Cara Menjalankan Aplikasi
1. Pastikan sudah menginstal **XAMPP** atau server lokal sejenis.
2. Salin folder project ini ke dalam direktori: htdocs/
3. Aktifkan **Apache** dan **MySQL** melalui XAMPP Control Panel.
4. Import file database `db_restoran.sql` ke MySQL melalui **phpMyAdmin**.
5. Akses aplikasi melalui browser:
   - http://localhost/restoran/login.php
   - http://localhost/restoran/pesanan.php

## Catatan
- File database (`db_restoran.sql`) disertakan sebagai pendukung untuk menjalankan aplikasi secara penuh seperti pada saat demo.
